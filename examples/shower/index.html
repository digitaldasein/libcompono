The dd-present template

dd-present facilitates and harmonizes your slide composition by extending the Shower HTML presentation template — mainly by adding a configuration file, a custom style sheet , and two open web components. Respectively:

  • config.json, to provide meta data (author, title, date, ...)
  • dd_basic.css, to tweak your presentation layout (colors, font-sizes, margins, ...)
  • <dd-slide>, to minimize boilerplate code in slide building
  • <dd-grid>, to minimize boilerplate code in grid building

The configuration file

The presentation caption banner, title slide, and footer automagically nest the information from the configuration file. Therefore, you need to change its data in ./config.json, which default to:

{
"author" : "Gerben Peeters & Senne Van Baelen",
"title" : "Welcome to digital dasein!",
[...],
"date" : "2022-02-24",
"company" : "Digital Dasein"
}
